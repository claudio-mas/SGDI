## Ícones Dinâmicos (Jinja2)

```html
<!-- Ícone condicional -->
<i class="bi {{ 'bi-check-circle-fill' if status == 'aprovado' else 'bi-x-circle-fill' }}"></i>

<!-- Ícone com classe dinâmica -->
<i class="bi bi-{{ icon_name }}"></i>

<!-- Múltiplas condições -->
<i class="bi {{ 'bi-clock' if status == 'pendente' else ('bi-check-circle-fill' if status == 'aprovado' else 'bi-x-circle-fill') }}"></i>
```

## Dicas de Uso

1. Use classes `me-*` (margin-end) para espaçamento após ícones em textos
2. Use `fs-*` para controlar o tamanho dos ícones
3. Combine com classes de cor do Bootstrap para indicar status
4. Em botões pequenos, considere usar apenas o ícone sem texto
5. Sempre adicione `title` em botões que só têm ícones para acessibilidade
