Sobre a Busca em Conteúdo

A busca em conteúdo permite pesquisar dentro do texto dos documentos PDF.

Como funciona:
  • O sistema extrai e indexa o texto de documentos PDF durante o upload
  • A busca procura por palavras ou frases dentro do conteúdo dos documentos
  • Apenas documentos que você possui ou que foram compartilhados com você são pesquisados
  • Os resultados são ordenados por relevância e data de upload
Dicas:
  • Use palavras-chave específicas para melhores resultados
  • Coloque frases entre aspas para busca exata: "contrato de prestação"
  • A busca não diferencia maiúsculas de minúsculas
  • Apenas documentos PDF com texto extraível são indexados
Nota: A busca em conteúdo requer que o SQL Server Full-Text Search esteja configurado. Se não estiver disponível, o sistema usará a busca simples como alternativa.
